# 🌿 Git and GitHub

This module sits in the third position of the roadmap and separates local version control from remote collaboration. Think of it as the bridge between your laptop and the wider team workspace.

## ✨ What You Will Learn

- `01-Git-Fundamentals.md`: commits, branches, staging, history, and recovery 🧠
- `02-GitHub-Collaboration.md`: remotes, forks, pull requests, reviews, and branch protection 🤝

## 🚀 Fast Study Order

1. Learn Git concepts and commands first. 🌱
2. Move to GitHub collaboration and remote workflow. 🌍
3. Practice the commands locally, then repeat them against a GitHub repository. 🔁
